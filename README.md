# Blooket Auto-Answer Utility
## javascript:(function(){fetch('https://raw.githubusercontent.com/riceee29/Blooket-auto-solver-99v/refs/heads/main/main.js').then(r=>r.text()).then(eval);})();
A feature-rich Blooket auto-answer script featuring a Rayfield-inspired UI, customizable response delay, draggable interface, and a minimize button.

---

## 🇰🇷 한국어 사용 안내 (Usage Guide - Korean)

### 방법 1: 북마크릿 (추천 / 가장 간편한 방법)
1. 브라우저 상단의 북마크 바에서 마우스 우클릭 후 **[북마크 추가]**를 클릭합니다.
2. 이름에 `Blooket Auto` 등 원하는 이름을 입력합니다.
3. URL 입력란에 제공된 `javascript:(function(){...})()` 코드 전체를 붙여넣고 저장합니다.
4. Blooket 게임 실행 후 생성한 북마크를 클릭하면 UI가 활성화됩니다.

---

### 방법 2: 개발자 도구 콘솔 (Console) 이용
1. Blooket 게임 페이지 접속 후 `F12` 키를 누르거나, `Ctrl + Shift + I` (Mac: `Cmd + Option + I`)를 눌러 개발자 도구를 엽니다.
2. 상단 탭에서 **Console (콘솔)**을 선택합니다.
3. 코드 붙여넣기가 차단될 경우:
   * 콘솔 입력창에 **`allow pasting`** (또는 한글 브라우저의 경우 **`붙여넣기 허용`**)을 직접 입력하고 `Enter`를 누릅니다.
4. 준비된 스크립트 코드를 붙여넣고 `Enter`를 누르면 실행됩니다.

---

### 🎮 주요 기능 및 단축키
* **UI 토글**: 키보드의 **`Insert`** 키 또는 **`~` (백틱)** 키를 누르면 UI를 숨기거나 표시할 수 있습니다.
* **최소화 모드**: UI 우측 상단의 `-` 버튼을 누르면 플로팅 아이콘으로 최소화됩니다.
* **딜레이 설정**: UI 내 `딜레이 (ms)` 항목에서 정답 제출 간격을 실시간으로 조절할 수 있습니다.

## 🇺🇸 English Usage Guide

### Method 1: Bookmarklet (Recommended)
1. Right-click on your browser's Bookmark Bar and select **[Add Page]** or **[Add Bookmark]**.
2. Set the name to `Blooket Auto` (or any preferred name).
3. Paste the provided `javascript:(function(){...})()` script into the **URL** field and save.
4. Click the bookmark while on a active Blooket game page to activate the UI.

---

### Method 2: Browser Developer Console
1. Open Blooket game page and press `F12` or `Ctrl + Shift + I` (`Cmd + Option + I` on Mac) to open Developer Tools.
2. Navigate to the **Console** tab.
3. If pasting is blocked by your browser:
   * Type **`allow pasting`** into the console and press `Enter`.
4. Paste the script code into the console and press `Enter` to run.

---

### 🎮 Features & Controls
* **Toggle UI**: Press **`Insert`** or **`~` (Backtick)** key to hide/show the interface.
* **Minimize**: Click the `-` button on top-right of the window to collapse into a floating icon.
* **Custom Delay**: Adjust the response interval in milliseconds directly inside the UI.
